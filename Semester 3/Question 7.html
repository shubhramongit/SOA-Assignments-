Lists

Lists are used to group related content. HTML supports ordered and unordered lists.

Ordered List
A list where items are numbered (e.g., 1, 2, 3).
Unordered List
A list where items are marked with bullets (e.g., •, ◦).

Linking

Links allow navigation between pages or sections within a document.

Anchor Tags
Used to create hyperlinks using the tag.
Internal Links
Links pointing to sections within the same page.

Formatting

HTML offers several formatting elements to style text content.

Bold
Use the or tags to make text bold.
Italic
Use the or tags to italicize text.

Selectors

CSS selectors are used to target and style HTML elements.

Class Selector
Selects elements with a specific class attribute.
ID Selector
Selects a single element with a specific ID.

Pseudo-Classes

Pseudo-classes define the special state of an element.

:hover
Applies styles when the user hovers over an element.
:focus
Applies styles when an element is focused.
